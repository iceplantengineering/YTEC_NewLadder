## 1. タイムアウト監視機能（LAD_100）

### 1.1 概要

各処理にタイムアウト監視を追加し、無限待機を防止します。

### 1.2 タイムアウト値設定

| 処理項目 | 標準時間 | タイムアウト値 | タイマー番号 |
|----------|----------|---------------|-------------|
| PC通信 | 1秒 | 5秒 | T100 |
| スタッカー通信 | 0.5秒 | 3秒 | T101 |
| B01通信 | 0.5秒 | 3秒 | T102 |
| B02通信 | 0.5秒 | 3秒 | T103 |
| 搬入タスク処理 | 60秒 | 120秒 | T110 |
| 搬出タスク処理 | 60秒 | 120秒 | T111 |
| 移庫タスク処理 | 90秒 | 180秒 | T112 |

## 2. 二重要求チェック機能（LAD_110）

### 2.1 概要

同一タスクの重複要求をチェックし、デッドロックを防止します。

### 2.2 変数定義

| 変数名 | アドレス | 内容 |
|--------|----------|------|
| D10 | D10 | 現在実行中タスク種別（0:なし、1:搬入、2:搬出、3:移庫） |
| D11 | D11 | 前回要求タスク種別 |
| D20 | D20 | 現在処理中RFIDコード |
| D21 | D21 | 新規要求RFIDコード |
| M400 | M400 | 二重要求検出フラグ |

### 2.3 ラダープログラム

#### 2.3.1 搬入要求二重チェック

```
┌─────────────────────────────────────────────────────────────────┐
│ LAD_110-1: 搬入要求二重チェック                                 │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│  [X0: 搬入要求信号入力]                                          │
│   └──[CMP D10, K1]──→ M401: 搬入実行中判定                     │
│          (現在タスク = 搬入?)                                    │
│                                                                  │
│  [X0: 搬入要求信号入力]                                         │
│   └──[CMP D21, D20]──→ M402: 同一RFID判定                       │
│          (新規RFID = 処理中RFID?)                                │
│                                                                  │
│  M401: 搬入実行中                                               │

│   └──[SET M400: 二重要求検出]                                   │
│   └──[SET M220: 二重搬入エラー]                                 │

│   └──[SET M310: 二重要求エラー通知]                             │
│                                                                  │
│  [X0: 搬入要求信号入力]                                         │

│   └──[MOV D10, K1]→ タスク種別設定（搬入）                      │
```

## 3. デッドロック検出・解除機能（LAD_120）

### 3.1 概要

全装置の状態を監視し、デッドロックを検出・解除します。

### 3.2 デッドロック検出条件

| 条件 | 内容 | 検出方法 |
|------|------|----------|
| 条件1 | 全装置が同一処理で一定時間以上変化なし | 全処理タイムアウト |
| 条件2 | タイムアウトエラーが複数装置で同時発生 | タイムアウトカウンター |
| 条件3 | 状態整合性エラーが検出 | 状態不一致フラグ |

### 3.3 変数定義

| 変数名 | アドレス | 内容 |
|--------|----------|------|
| D30 | D30 | タイムアウト発生カウンター |
| D31 | D31 | デッドロック検出カウンター |
| M500 | M500 | デッドロック検出フラグ |
| M501 | M501 | 自動解除可能フラグ |
| M502 | M502 | 手動解除要求フラグ |
| M510 | M510 | デッドロック警報出力 |
