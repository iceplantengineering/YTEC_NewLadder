## 1. 二重要求チェック機能（LAD_110）【改訂】

### 1.1 変更後変数定義

| 変数名 | アドレス（変更後） | 内容 |
|--------|-------------------|------|
| D610 | D610 | 現在実行中タスク種別 |
| D611 | D611 | 新規要求タスク種別 |
| D620 | D620 | 現在処理中RFIDコード |
| D621 | D621 | 新規要求RFIDコード |

### 1.2 ラダープログラム【改訂】

```
┌─────────────────────────────────────────────────────────────────┐
│ LAD_110-1: 搬入要求二重チェック（改訂）                           │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│  [X0: 搬入要求信号入力]                                        │
│   └──[CMP D610, K1]──→ 搬入実行中判定                           │
│          (現在タスク = 搬入?)                                     │
│                                                                  │
│  [X0: 搬入要求信号入力]                                        │
│   └──[CMP D621, D620]──→ 同一RFID判定                           │
│          (新規RFID = 処理中RFID?)                                  │
│                                                                  │
│  搬入実行中 + 同一RFID                                         │
│   └──[SET 二重要求検出]                                          │
│   └──[SET 二重搬入エラー]                                        │
│   └──[RST X0: 搬入要求信号]                                     │
│   └──[SET 二重要求エラー通知]                                   │
│                                                                  │
│  [X0: 搬入要求信号入力]                                        │
│  + 二重要求未検出                                                 │
│   └──[MOV D610, K1]→ タスク種別設定（搬入）                      │
│   └──[MOV D621, D620]→ RFIDコード保存                           │
│   └──[SET 搬入タスク開始]                                        │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

## 2. タイムアウト監視機能（LAD_100）【変更なし】

A01のタイムアウト監視機能は、Mアドレスの競合がないため、変更なしでそのまま使用可能です。

---

## 3. デッドロック検出・解除機能（LAD_120）【変更なし】

A01のデッドロック検出・解除機能は、Mアドレスの競合がないため、変更なしでそのまま使用可能です。

---
