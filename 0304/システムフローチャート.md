## 4. 搬入シーケンス詳細フロー

```mermaid
flowchart TB
    START([開始]) --> RECV_TASK[PCよりタスク受信 搬入指令 棚位置]
    RECV_TASK --> SEL_CONV[搬入台選定 CV1 or CV3]
    SEL_CONV --> CHECK_PERM{搬入許可確認 B01搬入可信号}
    CHECK_PERM -->|許可| SEND_TASK[タスク送信 B01搬入タスクID]
    CHECK_PERM -->|不可| END([完了])

    SEND_TASK --> WAIT_AGV[AGV到着待ち 搬入AGV在席確認]
    WAIT_AGV --> READ_RFID[RFID読み取り パレットID確認]
    READ_RFID --> RUN_CV[搬入CV運転 パレット搬入]
    RUN_CV --> CHECK_PAL{搬入完了確認 フォーク部在席}
    CHECK_PAL -->|完了| NOTIFY_A01[B01完了通知]
    CHECK_PAL -->|未完了| RUN_CV

    NOTIFY_A01 --> CMD_STK[スタッカーへ搬入指示 タスク送信]
    CMD_STK --> STK_ACTION[スタッカー搬入動作 受取 格納]
    STK_ACTION --> COMPLETE[搬入完了 タスク完了報告]
    COMPLETE --> END

    style START fill:#c8e6c9
    style END fill:#c8e6c9
    style RECV_TASK fill:#e1f5fe
    style STK_ACTION fill:#ffe1f5
    style CHECK_PERM fill:#fff9c4
    style CHECK_PAL fill:#fff9c4
```

## 5. 搬出シーケンス詳細フロー

```mermaid
flowchart TB
    START([開始]) --> RECV_TASK[PCよりタスク受信 搬出指令 棚位置]
    RECV_TASK --> CMD_STK[スタッカーへ搬出指示 タスク送信]
    CMD_STK --> STK_ACTION[スタッカー搬出動作 取出 移動]
    STK_ACTION --> SEL_CONV[出庫台選定 CV2 or CV4]
    SEL_CONV --> CHECK_READY{搬出台準備完了 B02搬出可信号}
    CHECK_READY -->|完了| SEND_TASK[タスク送信 B02搬出タスクID]
    CHECK_READY -->|未完了| RETRY[リトライ]
    RETRY --> SEL_CONV

    SEND_TASK --> TRANSFER[受渡動作 スタッカー CV]
    TRANSFER --> READ_RFID[RFID読み取り パレットID照合]

    CHECK_DEP -->|可| RUN_CV[搬出CV運転 パレット搬出]
    CHECK_DEP -->|不可| WAIT[待機]
    WAIT --> SET_DEST

    RUN_CV --> COMPLETE[搬出完了 タスク完了報告]
    COMPLETE --> END([完了])

    style START fill:#c8e6c9
```

## 6. 入庫側コンベア（B01）制御フロー

```mermaid
flowchart TB
    STANDBY([待機状態])

    STANDBY --> DECIDE{要求判定}

    DECIDE -->|搬入要求| REQ_IN[搬入要求受信 搬入タスク]
    DECIDE -->|搬出要求| REQ_OUT[搬出要求受信 搬出タスク]
    DECIDE -->|エラー発生| ERROR[エラー発生]

    REQ_IN --> CHECK_AGV_IN[AGV到着確認 在席確認]
    CHECK_AGV_IN --> RFID_IN[RFID読取実行 パレットID]
    RFID_IN --> RUN_IN[搬入CV運転 減速 停止]
    RUN_IN --> CHECK_PAL_IN{フォーク部在席確認}
    CHECK_PAL_IN -->|在席| NOTIFY_IN[完了通知送信 A01へ]
    CHECK_PAL_IN -->|未着| RUN_IN
    NOTIFY_IN --> LOOP1[ループ]

    REQ_OUT --> PREP_OUT[搬出台準備 フォーク位置]
    PREP_OUT --> WAIT_STK[スタッカー準備待機]
    WAIT_STK --> TRANS_OUT[受渡実行 スタッカー]
    TRANS_OUT --> RUN_OUT[搬出CV運転 AGVへ搬出]
    RUN_OUT --> NOTIFY_OUT[完了通知送信 A01へ]
    NOTIFY_OUT --> LOOP2[ループ]

    ERROR --> STOP[エラー停止 異常表示 停止信号]

    LOOP1 --> STANDBY
    LOOP2 --> STANDBY

    style STANDBY fill:#c8e6c9
    style ERROR fill:#ffcdd2
    style STOP fill:#ffcdd2
```

### 6.1 主要センサー一覧

| センサー名 | 機能 |
|-----------|------|
| 搬入CV減速・停止 | 搬入パレット位置検出 |
| 搬出CV減速・停止 | 搬出パレット位置検出 |
| 搬入/搬出フォーク上・下降端 | フォーク位置確認 |
| 搬入/搬出台上・下降端 | リフター位置確認 |
| 搬入/搬出フォーク部パレット在席 | パレット有無 |
| AGV在席検出 | AGV到着確認 |
| RFID読取値 | パレットID確認 |
| LC正常 | リミットスイッチ正常確認 |

### 8.1 スタッカー軸構成

| 軸 | 機能 | センサー |
|----|------|----------|

- 各軸速度
- 各軸トルク
- 各軸アラーム状態
- フォーク状態（受取中/受渡中/完了）

## 14. 装置IO一覧（抜粋）

### A01 主な入出力信号

**入力信号:**
| 信号 | 内容 |
|------|------|
| X1000-X102F | B01/B02入力信号 |
| M1036 | 搬入フォーク部パレット在席 |
| M1052-M1053 | 搬出CV停止・減速 |

**出力信号:**
| 信号 | 内容 |
|------|------|
| Y1000-Y102F | B01/B02出力信号 |
| Y102A | 搬入受取可 |
| Y102B | 搬出受渡可 |

### A02 主な入出力信号

**入力信号:**
| 信号 | 内容 |
|------|------|
