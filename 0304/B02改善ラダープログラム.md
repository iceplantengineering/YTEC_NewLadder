## 1. タイムアウト監視機能（LAD_400）

### 1.1 概要

各処理にタイムアウト監視を追加し、無限待機を防止します。

### 1.2 タイムアウト値設定

| 処理項目 | 標準時間 | タイムアウト値 | タイマー番号 |
|----------|----------|---------------|-------------|
| CV1搬送 | 10秒 | 20秒 | T400 |
| CV2搬送 | 10秒 | 20秒 | T401 |
| Jacking昇降 | 5秒 | 15秒 | T402 |
| フォーク動作 | 5秒 | 15秒 | T403 |
| AGV到着待ち | 30秒 | 60秒 | T410 |
| 出庫処理全体 | 60秒 | 120秒 | T420 |

### 1.3 変数定義

| 変数名 | アドレス | 内容 |
|--------|----------|------|
| M1200 | M1200 | CV1搬送中 |
| M1201 | M1201 | CV2搬送中 |
| M1202 | M1202 | Jacking昇降中 |
| M1203 | M1203 | フォーク動作中 |
| M1210 | M1210 | AGV到着待ち |
| M1220 | M1220 | 出庫処理中 |
| M1300 | M1300 | CV1タイムアウト |
| M1301 | M1301 | CV2タイムアウト |
| M1302 | M1302 | Jackingタイムアウト |
| M1303 | M1303 | フォークタイムアウト |
| M1310 | M1310 | AGV到着タイムアウト |
| M1320 | M1320 | 出庫タイムアウト |

## 2. 状態保存・復元機能（LAD_410）

### 2.1 概要

処理中断時の状態を不揮発性領域に保存し、再開時に復元します。

### 2.2 保存項目

| 項目 | 保存先アドレス | 内容 |
|------|---------------|------|
| 実行中工程 | D400 | 実行中工程番号 |
| CV1状態 | D401 | CV1状態 |
| CV2状態 | D402 | CV2状態 |
| Jacking状態 | D403 | Jacking状態 |
| フォーク状態 | D404 | フォーク状態 |
| AGV状態 | D405 | AGV状態 |
| エラーコード | D406 | 発生エラーコード |
| 保存時刻 | D407 | 状態保存時刻 |

### 2.3 変数定義

| 変数名 | アドレス | 内容 |
|--------|----------|------|
| M1500 | M1500 | 状態保存要求 |
| M1501 | M1501 | 状態保存完了 |
| M1510 | M1510 | 状態復元要求 |
| M1511 | M1511 | 状態復元完了 |
| D410 | D410 | 保存データ整合性チェック用 |

### 2.4 ラダープログラム

## 3. 二重要求チェック機能（LAD_420）

### 3.1 概要

出庫要求の重複をチェックし、デッドロックを防止します。

### 3.2 変数定義

| 変数名 | アドレス | 内容 |
|--------|----------|------|
| D420 | D420 | 現在処理中出庫要求ID |
| D421 | D421 | 新規出庫要求ID |
| D422 | D422 | 前回出庫要求ID |
| M1600 | M1600 | 出庫処理中フラグ |
| M1601 | M1601 | 二重要求検出フラグ |

### 3.3 ラダープログラム

#### 3.3.1 出庫要求受信処理

```
┌─────────────────────────────────────────────────────────────────┐
│ LAD_420-1: 出庫要求受信処理                                     │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│  [X40: 出庫要求信号入力]                                        │
│   └──[MOV D421, 要求ID]→ 新規要求ID保存                        │
│                                                                  │
│  [X40: 出庫要求信号入力]                                        │
│  + [M1600: 出庫処理中]                                          │
│   └──[CMP D421, D420]──→ M1601: 二重要求検出                  │
│          (新規要求ID = 処理中要求ID?)                            │
│                                                                  │
│  M1601: 二重要求検出                                            │
│   └──[SET M1620: 二重出庫エラー]                                │
│   └──[RST X40: 出庫要求信号]                                    │
│   └──[SET Y162: 二重エラー表示]                                 │
│   └──[SET M1460: 二重要求エラー通知]                           │
│   └──[RST M1601: 二重要求検出]                                  │
│                                                                  │
│  [X40: 出庫要求信号入力]                                        │
│  + M1601/: 二重要求未検出                                       │
│  + M1600/: 出庫処理中でない                                     │
│   └──[SET M1600: 出庫処理中]                                    │
│   └──[MOV D420, D421]→ 処理中要求ID設定                        │
│   └──[MOV D422, D421]→ 前回要求ID保存                          │
│   └──[SET M1610: 出庫タスク開始]                                │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

#### 3.3.2 出庫完了時リセット

```
┌─────────────────────────────────────────────────────────────────┐
│ LAD_420-2: 出庫完了時リセット                                   │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│  [M1611: 出庫完了信号]                                         │
│   └──[RST M1600: 出庫処理中]                                    │
│   └──[RST M1610: 出庫タスク開始]                                │
│   └──[MOV D420, K0]→ 処理中要求IDクリア                         │
│   └──[RST M1620: 二重エラー]                                    │
│   └──[RST Y162: 二重エラー表示]                                 │
│   └──[RST M1460: 二重要求エラー通知]                           │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```
