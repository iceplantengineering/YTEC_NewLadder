## 1. タイムアウト監視機能（LAD_300）

### 1.1 概要

各軸・各処理にタイムアウト監視を追加し、無限待機を防止します。

### 1.2 タイムアウト値設定

| 処理項目 | 標準時間 | タイムアウト値 | タイマー番号 |
|----------|----------|---------------|-------------|
| X軸走行 | 30秒 | 60秒 | T300 |
| Y軸昇降 | 20秒 | 40秒 | T301 |
| Z軸フォーク | 5秒 | 15秒 | T302 |
| 原点復帰 | 45秒 | 90秒 | T310 |
| 搬送処理 | 25秒 | 50秒 | T320 |
| 搬出処理 | 25秒 | 50秒 | T321 |
| 移庫処理 | 50秒 | 100秒 | T322 |

| M710 | M710 | 原点復帰中 |
| M720 | M720 | 搬送処理中 |
| M721 | M721 | 搬出処理中 |
| M722 | M722 | 移庫処理中 |
| M800 | M800 | X軸タイムアウト |
| M801 | M801 | Y軸タイムアウト |

## 2. 状態保存・復元機能（LAD_310）

### 2.1 概要

処理中断時の状態を不揮発性領域に保存し、再開時に復元します。

### 2.2 保存項目

| 項目 | 保存先アドレス | 内容 |
|------|---------------|------|
| 現在X軸位置 | D300 | 現在のX軸位置 |
| 現在Y軸位置 | D301 | 現在のY軸位置 |
| 現在Z軸位置 | D302 | 現在のZ軸位置 |
| 実行中タスク | D303 | 実行中タスク種別 |
| 実行中工程 | D304 | 実行中工程番号 |
| フォーク状態 | D305 | フォーク状態 |
| エラーコード | D306 | 発生エラーコード |
| 保存時刻 | D307 | 状態保存時刻 |

### 2.3 変数定義

| 変数名 | アドレス | 内容 |
|--------|----------|------|
| M1000 | M1000 | 状態保存要求 |
| M1001 | M1001 | 状態保存完了 |
| M1010 | M1010 | 状態復元要求 |
| M1011 | M1011 | 状態復元完了 |
| D310 | D310 | 保存データ整合性チェック用 |

### 2.4 ラダープログラム

#### 2.4.1 状態保存処理

```
┌─────────────────────────────────────────────────────────────────┐
│ LAD_310-1: 状態保存処理                                         │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │

│   └──[SET M1000: 状態保存要求]                                  │
│                                                                  │
│  [非常停止信号]                                                  │
│   └──[SET M1000: 状態保存要求]                                  │
│                                                                  │
│  M1000: 状態保存要求                                           │
│   └──[MOV D300, 現在X軸位置]→ X軸位置保存                      │
│   └──[MOV D301, 現在Y軸位置]→ Y軸位置保存                      │
│   └──[MOV D302, 現在Z軸位置]→ Z軸位置保存                      │
│   └──[MOV D303, 実行中タスク]→ タスク保存                       │
│   └──[MOV D304, 実行中工程]→ 工程保存                           │
│   └──[MOV D305, フォーク状態]→ フォーク状態保存                 │
│   └──[MOV D306, エラーコード]→ エラーコード保存                 │
```
