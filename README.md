# YTEC 自動倉庫システム（スタッカークレーン）

自動倉庫システムのPLCプログラムおよび仕様書を管理するリポジトリです。

## 📋 概要

本システムは、スタッカークレーンを用いた自動倉庫システムの制御プログラムおよび関連仕様書を含んでいます。

### システム構成

| 装置 | 説明 |
|------|------|
| **A01** | 地上側自動倉庫制御盤（マスタ局） |
| **A02** | スタッカークレーン本体 |
| **B01** | 入庫側コンベア制御盤 |
| **B02** | 出庫側コンベア制御盤 |
| **AGV** | AGV統括盤 |

### 通信プロトコル

| 経路 | プロトコル |
|------|-----------|
| PC ↔ A01 | Modbus TCP/IP |
| A01 ↔ A02 | 内部BMOV |
| A01 ↔ B01/B02/AGV | CC-Link IE |

## 📊 システムフローチャート

## 🔧 タスク種別

| タスクタイプ | 説明 |
|-------------|------|
| **Type: 1** | 搬入タスク（AGV/搬入台 → 入庫CV → スタッカー → 棚格納） |
| **Type: 2** | 搬出タスク（棚 → スタッカー → 出庫CV → AGV/搬出台） |
| **Type: 3** | 移庫タスク（棚A → スタッカー → 棚B） |
| **Type: 4** | 原位置復帰（スタッカーを原点位置へ復帰） |

## 🚀 運転モード

| モード | 説明 |
|--------|------|
| 自動/連動 | PC連携運転（PCからのタスク実行、最適化制御） |
| 自動/単独 | 単独運転（A01のみでのタスク管理・実行） |
| 手動 | 手動運転（各軸個別操作、試運転用） |
| 点検 | 点検モード（メンテナンス用、制限速度） |
